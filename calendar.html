Calendar Docs

Copy JavaScript for Calendar.

Add CDN to your code above Calendar rendering script.

<script src="https://www.andrejvysny.online/widgets/calendar/calendar.js"> </script>

Add Calendar style to your code in head.

<link src="https://www.andrejvysny.online/widgets/calendar/calendar.css" rel="stylesheet" type="text/css">

Render your calendar.

            
                new Calendar({
                    root: 'root',
                });

Options

  • days - names for days in week. default: ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']
  • months - names for months in year. default: ['Jan', 'Feb', 'Mar', 'Apr', 'May', 'June', 'July', 'Aug', 'Sept', 'Oct', 'Nov', 'Dec']
  • renderLast - render cells after last date to complete Calendar table. default: true
  • autoInit - automatic Calendar initialization after creating. default: true
  • initMonth - month showed after initialization. default: *current Month
  • initYear - year showed after initialization. default: *current Year
  • selectable - allow you to select date in calendar by adding attribute "selected" to selected date. default: false
  • minDate - set minimum showed day (erlier dates are disabled). Enter date in "2020-7-22" format or for today's date "today". default: null
  • maxDate - set maximum showed day (later dates are disabled). Enter date in "2020-7-22" format or for today's date "today". default: null

Functions

  • init() - Initialize calendar (use if autoInit is set to false).
    let calendar = new Calendar({
        root: 'root',
        autoInit: false,
    });

    calendar.init(); // Now you will initialize calendar.
  • updateMonth() - switch Calendar to entered month and year.
    let calendar = new Calendar({
        root: 'root'
    });

    // Now you will change month in calendar to April and year to 2020.
    calendar.updateMonth(2020,4);
